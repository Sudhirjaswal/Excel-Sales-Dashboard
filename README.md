# 📊 Sales Dashboard (Excel Project)

## 🔍 Overview
This project is an interactive Excel dashboard built to analyze sales data and answer key business questions.

## 🎯 Objectives
- Analyze monthly sales & orders
- Identify top-performing states
- Compare purchasing behavior by gender
- Understand order status distribution
- Find highest contributing sales channels
- Analyze age vs gender patterns

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Visualizations
- Data Cleaning

## 📈 Key Insights
- Identified peak sales months
- Top 5 states contributing to revenue
- Gender-based purchasing trends
- Most used sales channel
